# Installation

```sh
$ git clone https://github.com/sobchenyuk/HTML5-Boilerplate-and-Bulma.io.git
$ cd HTML5-Boilerplate-and-Bulma.io.git
$ npm install
```

# Gulp

```sh
$ gulp default
```
